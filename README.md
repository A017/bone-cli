# bone-cli
> bone的命令行工具

###安装
通过npm安装，这是全局模块，安装后可以在命令行中使用`bone`命令

```sh
npm install -g bone-cli
```

bone-cli会载入你项目目录下的[bone](https://github.com/wyicwx/bone)模块，并给bone对象上添加commander对象(commander对象是[Commander](https://github.com/tj/commander.js)的一个实例)
